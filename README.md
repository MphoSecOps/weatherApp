# 🌦️ WeatherAPP

A modern and responsive weather application built with **HTML**, **CSS**, and **JavaScript** using the OpenWeatherMap.

The app allows users to search for South African cities and view real-time weather information including:

* 🌡️ Temperature
* 💧 Humidity
* 🌬️ Wind Speed
* ☁️ Weather Conditions
* 🌍 Dynamic Weather Icons

---

## 🚀 Features

* Real-time weather data
* South Africa city search support
* Dynamic weather icons
* Error handling for invalid cities
* Responsive modern UI
* Simple and lightweight frontend project

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* OpenWeatherMap API
